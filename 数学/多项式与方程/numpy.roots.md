当然可以！我们来通过一个具体的例子，使用 Python 的 `numpy.roots()` 函数来求解一元二次方程：

---

## 📘 一元二次方程回顾

标准形式为：
$$
ax^2 + bx + c = 0
$$

其中 $ a \neq 0 $。  
根据判别式 $ D = b^2 - 4ac $，可以判断根的类型：

- $ D > 0 $：两个不同的实数根
- $ D = 0 $：一个重根（相同实数根）
- $ D < 0 $：一对共轭复数根

---

## ✅ 使用 `numpy.roots()` 求解

`numpy.roots([a, b, c])` 接受一个系数数组 `[a, b, c]`，返回方程的所有根。

---

### 🔢 示例1：有两个实数根

求解方程：
$$
x^2 - 5x + 6 = 0
$$

```python
import numpy as np

# 系数 [a, b, c]
coeffs = [1, -5, 6]

# 求根
roots = np.roots(coeffs)

print("根为：", roots)
```

**输出：**
```
根为： [3. 2.]
```

✅ 解释：这个方程有两个实数根：2 和 3。

---

### 🔢 示例2：有一个实数重根（D = 0）

求解方程：
$$
x^2 - 4x + 4 = 0
$$

```python
coeffs = [1, -4, 4]
roots = np.roots(coeffs)
print("根为：", roots)
```

**输出：**
```
根为： [2. 2.]
```

✅ 解释：这是一个重根，两个相等的实数根都是 2。

---

### 🔢 示例3：有一对共轭复数根（D < 0）

求解方程：
$$
x^2 + x + 1 = 0
```

```python
coeffs = [1, 1, 1]
roots = np.roots(coeffs)
print("根为：", roots)
```

**输出：**
```
根为： [-0.5+0.8660254j -0.5-0.8660254j]
```

✅ 解释：得到一对共轭复数根。

---

## 📌 小结

| 类型            | 判别式 $ D $ | 根的类型             | 示例                |
|-----------------|---------------|------------------------|---------------------|
| 两个不同实根    | $ D > 0 $   | 实数                   | $ x^2 -5x +6=0 $     |
| 一个重根         | $ D = 0 $   | 相同实数               | $ x^2 -4x +4=0 $     |
| 一对共轭复数根   | $ D < 0 $   | 共轭复数               | $ x^2 +x +1=0 $      |

如果你想手动实现公式求解，也可以这样写：

```python
import cmath  # 支持复数计算

a, b, c = 1, 1, 1
d = b**2 - 4*a*c

root1 = (-b + cmath.sqrt(d)) / (2*a)
root2 = (-b - cmath.sqrt(d)) / (2*a)

print("根为：", root1, root2)
```
